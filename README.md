# Hábitos Rubén

Aplicación personalizada de seguimiento de hábitos diseñada para acompañar rutinas diarias de forma sencilla, visual y motivadora.

## Objetivo

El objetivo de esta app es ayudar al usuario a mantener constancia en sus hábitos personales mediante una interfaz clara, directa y adaptada a sus necesidades.

Forma parte de una serie de aplicaciones personalizadas creadas como práctica de desarrollo web y como exploración del enfoque boutique de Vitalab: herramientas pequeñas, concretas y diseñadas para personas reales.

## Funcionalidades principales

- Seguimiento de hábitos diarios.
- Interfaz sencilla y orientada al uso cotidiano.
- Diseño personalizado para el usuario final.
- Enfoque motivacional para reforzar la constancia.
- Proyecto web ligero y fácil de desplegar.

## Tecnologías utilizadas

- HTML
- CSS
- JavaScript

## Estado del proyecto

Proyecto funcional en fase inicial de aprendizaje y mejora continua.

## Aprendizajes trabajados

Con este proyecto se practican conceptos básicos de desarrollo web como:

- Estructura de una aplicación con HTML.
- Estilos visuales con CSS.
- Interactividad con JavaScript.
- Organización de una app pequeña orientada a un caso de uso real.
- Creación de proyectos personalizados como parte del aprendizaje práctico.
